Setting up a RethinkDB AMI: Choose a password

To initialize the AMI, please chose a password. This password will be used to access the web UI and as an authentication key for client drivers.

If you want this instance to connect to an existing cluster, enter one or more host:port values here, separated by spaces. Otherwise leave this field blank.

Extra documentation about this AMI can be found here: http://rethinkdb.com/todo/doc/ami
